Odoo ViettelPost Connector

The Odoo ViettelPost Connector module is an integrated product between the Odoo application and the ViettelPost carrier. The application provides features, through the API manipulated directly into the dashboard of ViettelPost.

  • Login and get token
  • Test connection
  • Provinces data synchronous
  • Districts data synchronous
  • Wards data synchronous
  • Post offices data synchronous
  • Services data synchronous
  • Extend services data synchronous
  • Warehouses data synchronous
  • Configuration warehouse ViettelPost from Odoo
  • Automatically generated a list of product types
  • Automatically generated a list of national types
  • Automatically generated a list of waybill types
  • Automatically generated a list of status
  • Create the waybill
  • Update status waybill
  • Automatically generated a list of national types
  • Print and save the waybill
  • Tracking log connect API

  1. Getting started

    • Install package selenium version 4.6.0

    • Install module Odoo ViettelPost Connector

  2. Login and get token

    Summary: Login and get the token. This token will help authenticate API connections with ViettelPost.

    Path: ViettelPost/Configuration/ViettelPost API Technical/Connection Config

    Note: (1) & (2): The username and password registed with viettelpost

    Step 1: Enter username(1) and password(2). Press button Get token

  3. Test Connection

    Summary: Check connection with ViettelPost's host

    Path: ViettelPost/Configuration/ViettelPost API Technical/Connection History

    Step 1: Press button Test connection

  4. Address synchronous

    Summary: Synchronous the list of provinces, districts, wards from ViettelPost and save it in odoo. This is required to configure the correct address information in odoo and ensure no errors when connecting to ViettelPost.

    Step 1: Go to ViettelPost/Configuration/Province. Press button Sync Province

    Step 2: Go to ViettelPost/Configuration/District. Press button Sync District

    Step 3: Go to ViettelPost/Configuration/Ward. Press button Sync Ward

  5. Post offices synchronous

    Summary: Synchronous the list of post offices across the country to Odoo. It makes it easy to find the nearest post office at Odoo

    Path: ViettelPost/Post Office

    Step 1: Press button Sync Post Office

  6. Service synchronous

    Summary: Synchronous the list of services and extend services available at ViettelPost.

    Path: ViettelPost/Service

    Step 1: Press button Sync Service

    Step 2: Press button Sync Extend Service

  7. Warehouse synchronous

    Summary: Synchronous the list of repositories that you have configured on the ViettelPost dashboard.

    Path: ViettelPost/Warehouse

    Step 1: Press button Sync Warehouse

  8. Configuration warehouse ViettelPost from Odoo

    Summary: Help you configure directly and quickly a ViettelPost warehouse at Odoo.

    Path: ViettelPost/Warehouse

    Step 1: Press button Create

    Step 2: Enter warehouse information

  9. Create waybill

    Summary: Create a viettelpost bill of lading directly and quickly at Odoo.

    Step 1: Go to Sales/Products. Create and configure a product.

    Note: Weight, Depth, Height is required.

    Step 2: Goto Contacts. Create a customer and configure receiver address.

    Note: Street, Ward, District, Province, Phone is required.

    Advance: You can search more and select a ward. Odoo will auto fill district and province.

    Step 3: Visit Sale. Create a sale order.

    Step 3.1: Add a customer

    Step 3.2: Add a product

    Step 3.3 Action Confirm

    Step 3.4: Now the field Delivery Method will appear. Add a Delivery Method (Select ViettelPost)

    Step 3.5: Add a warehouse

    Step 3.6: Select product type

    Step 3.7: Select National type

    Step 3.8: Select Waybill type

    Step 3.9: Press button Get a list of services that match itinerary to receive services that match itinerary

    Step 3.10: Select the service and extension service that matches the list of services that match the itineraries obtained.

    Step 3.11: Check information and create waybill

    Step 3.12: Check information 3pls created

  10. Print and save waybill

    Summary: Print and save the image of the bill of lading information to odoo.

    Advance: You have 3 choices of bill of lading (A5, A6, A7)

    Step 1: Press button Print waybill

    Step 2: Choice bill of lading and press button

    Step 3: Check bill of lading printed

  11. Update status waybill

    Summary: We will use two main states: confirm and cancel the bill of lading.

    Note: These operations work on both orders and bills of lading

    1. Confirm waybill

      Step 1: Press button Confirm waybill

    2. Cancel waybill

      Step 1: Press button Cancel

  12. Tracking log connect api

    Summary: Track all connection requests with api ViettelPost.

    Path: ViettelPost/Configuration/ ViettelPost API Technical/Connection History
